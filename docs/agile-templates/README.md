# Agile Deliverables Templates

This folder contains the default template set for the iterative fullstack methodology.

## Folder Map

- `research/`
- `planning/`
- `development/`
- `sprints/`
- `governance/`

## Standard Rules

- Every document carries a version.
- Every sprint produces both a completion report and a feedback report.
- Planning updates and development-spec updates must reference the sprint that triggered the revision.
- When a document changes meaningfully, record:
  - version
  - date
  - reason for change
  - affected downstream artifacts
